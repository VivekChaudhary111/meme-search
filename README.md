# Meme Search - Hinglish Meme Finder

Arre yaar! Internet pe kitne saare memes hai, lekin sahi wala dhundna mushkil hai na? Isliye maine ye **Meme Search** banaya hai! 

## Kya hai ye project?

Ye ek AI-powered meme search engine hai jo tumhe help karega perfect memes dhundne mein. Just type karo kya chahiye, aur boom! Sab se relevant memes mil jayenge.

## Features (Kya kya mil raha hai)

- **Smart Search**: AI se powered search jo samajh jaata hai tumhara context
- **Image Recognition**: Memes ko dekh ke samajh jaata hai kya hai usme
- **Fast Results**: Bilkul jhatpat results milenge
- **Mobile Friendly**: Phone pe bhi perfectly chalega
- **Accurate Matching**: Exactly wahi memes jo tum dhund rahe ho
- **Completely Free**: Bilkul free hai, koi paisa nahi lagega

## Installation (Kaise setup kare)

### Prerequisites
Pehle ye sab install karna padega:
- Node.js (latest version)
- npm ya yarn
- Git

### Steps
```bash
# Repository clone karo
git clone https://github.com/yourusername/meme-search.git

# Folder mein jao
cd meme-search

# Dependencies install karo
npm install

# Environment variables setup karo
cp .env.example .env
# Ab .env file mein apni API keys add karo

# Development server start karo
npm run dev
```

Bas ho gaya! Ab browser mein `http://localhost:3000` pe jao.

## Usage (Kaise use kare)

1. **Search Box** mein type karo jo meme chahiye
   - Example: "crying cat", "this is fine dog", "woman yelling at cat"

2. **Filters** use karo agar specific type chahiye
   - Popular memes
   - Recent uploads
   - Category wise (funny, sad, motivational, etc.)

3. **Click** karo jo meme pasand aaye, directly download ho jayega

## Tech Stack (Kya use kiya hai)

- **Frontend**: React.js, Next.js, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **AI/ML**: OpenAI API, TensorFlow.js
- **Storage**: Cloudinary
- **Deployment**: Vercel
